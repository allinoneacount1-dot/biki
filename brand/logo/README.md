# BIKI — Brand / Logo

> the mark. the rules. the construction.

---

## Concept

**two watchers over a threshold.**

- the **two circles** are the two i's in BIKI — the watchful principle
- the **threshold line** beneath is the plane one crosses to enter
- vertical stacking (not horizontal) reads as **presence**, not face

the mark is geometric, deliberate, monochrome. it carries authority by restraint.

---

## Files

| file | purpose | when to use |
|------|---------|-------------|
| `mark.svg` | the symbol alone | favicon, app icon, watermark, small UI |
| `wordmark.svg` | "BIKI" text alone | long body, signatures, vertical-constrained spaces |
| `primary-lockup.svg` | mark stacked above wordmark | **default** — landing, decks, marketing |
| `horizontal-lockup.svg` | mark left, wordmark right | nav bars, headers, email signatures |
| `construction.svg` | grid + measurements | reference only — do not ship |
| `clear-space.svg` | clearance diagram | reference only |
| `min-sizes.svg` | size scale reference | reference only |
| `usage-rules.md` | the rules | read this before placing the mark anywhere |

## Quick Reference

```
viewBox (mark):        24 × 32
base unit (1u):        2.5 (the radius)
circles:               r=2.5u, stacked vertically, 5u apart center-to-center
threshold line:        12u wide × 0.4u tall
clear space:           1u on all sides
minimum (digital):     16 × 21 px
minimum (print):       8 mm tall
color:                 currentColor (inherits)
```
